白色背景，浅灰色分割线，柔和阴影，圆角卡片，绿色按钮，浅绿色选中状态，绿色在线状态点，整体干净、有呼吸感、可读性强。

正面视角，全屏桌面 UI 截图，16:9 比例，布局平衡，细节清晰，专业产品设计展示，高分辨率。
