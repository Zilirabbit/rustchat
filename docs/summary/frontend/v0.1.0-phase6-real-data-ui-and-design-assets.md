# RustChat Summary

## 版本信息

- 版本：`v0.1.0`
- 阶段：`phase6`
- 主题：前端真实数据最小展示、CSS 重构与设计资产入库
- 日期：`2026-05-10`

## 本次完成内容

本次将前端聊天页从静态高保真 mock 收敛为基于已有接口的真实数据最小展示，并重构 `main.css`，移除多层浅色/暗色/hover 覆盖规则，解决 hover 后组件变深色的问题。

### 前端体验

- 保留 `Rust Chat` 品牌和白绿视觉风格
- 左侧栏只展示真实会话，并按 `group` 与 `private` 分组
- 去除静态频道、Overview/Mentions/Threads 等无后端数据支撑的入口
- 中间消息区只渲染真实历史消息和实时消息
- 右侧栏在选中真实群聊时展示已有成员接口返回的群成员
- 保留私聊创建、群聊创建、添加群成员、退出群聊和 WebSocket 文本发送能力

### 样式重构

- 重写 `frontend/src/styles/main.css` 为单层 light theme 样式
- 移除旧 MongoDB 暗色变量和重复覆盖规则
- hover 状态不再改变主背景色或文字色
- message composer、发送按钮、会话项、用户结果、Leave group 等 hover 改为边框、阴影和轻微上浮反馈
- 保留 active 会话的浅绿色选中态

### 设计资产与生成说明

- 新增 `docs/design/PROMPT.md`，保留 MongoDB 风格三栏聊天 UI 的中英文生成提示
- 新增 `docs/design/DESIGN.md`，保留 MongoDB 风格色彩、字体、圆角、间距和组件 token
- 新增参考图 `docs/img/UI-preview.png`
- 新增 `docs/design/README.md`，说明如何组合 prompt、design token 和参考图生成视觉稿
- 当前实现以真实接口最小展示为准，未落地 prompt 中缺少后端支撑的静态频道、在线状态、reaction、link preview、typing indicator 等内容
