# RustChat

RustChat is a chat application project with a Rust backend and a Vue 3 frontend.

## Project Layout

```text
rustchat-server/
├─ backend/   # Rust backend service, database migrations, backend env examples
├─ frontend/  # Vue 3 + TypeScript + Vite frontend
└─ docs/      # Product, architecture, API, setup, design, and progress docs
```

## Local Development

Backend:

```bash
cd backend
cargo run
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Environment examples live in `backend/.env.example` and `frontend/.env.example`.

## Generated Artifacts

These directories are generated or runtime artifacts and are intentionally not part of the source layout:

- `backend/target/`
- `frontend/node_modules/`
- `frontend/dist/`
- `backend/uploads/`

They can be regenerated by building, installing dependencies, running the app, or uploading files during local development.
